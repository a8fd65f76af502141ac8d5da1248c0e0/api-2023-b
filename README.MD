# 2

## Запуск

Приложение прослушивает порт **8080**.
Документация доступна по адресу:
**http://localhost:8080/docs**

Для запуска необходимы:
- docker
- docker compose 

Запуск:

1. Создание .env-файла. Достаточно скопировать/переименовать **env.example**.
```
cat ./env.example > .env
```
2. Запуск приложения
```
docker compose up
```
